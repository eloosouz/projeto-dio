O que são tags de título?

(Voltar ao topo)

As tags <h1> a <h6> são usadas para definir cabeçalhos em uma página HTML.

Hierarquia dos títulos

<h1> → Título principal (deve aparecer apenas uma vez por página).

<h2> → Subtítulos (ex: "Como funciona o HTML").

<h3> a <h6> → Subseções (quanto maior o número, menor a importância visual).

"A hierarquia de títulos é essencial para acessibilidade e SEO. Nunca pule níveis (ex: evite h1 → h3 sem h2)."

Formatação de Texto

Exemplos de formatação:

  • Negrito com <strong>
  • Itálico com <i>
  • Sublinhado com <u>
  • Texto marcado com <mark>
  • Texto pequeno com <small>
  • Texto excluído com <del>
  • H2O (subscrito) e x2 (sobrescrito)
  • HTML com <abbr>
  • Texto com fonte específica (tag obsoleta)
